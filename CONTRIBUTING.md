# AIGestion Sovereign Contribution Guidelines

Welcome to the AIGestion Nexus. This project follows the **Sovereign Intelligence** standard.

## God-Mode Development Standards
1. **Zero-Trust Hardening**: No hardcoded secrets. All environment variables must be validated via Zod.
2. **Infrastructure-First**: Before making backend changes, ensure the Docker environment is stable and health checks pass.
3. **Pnpm Monorepo**: Use `pnpm` for all dependency management. Avoid using `npm` or `yarn` directly in subdirectories.
4. **Vibrant Aesthetics**: Frontend contributions must adhere to the high-performance, premium design system documented in the Nexus KI.

## Pull Request Process
- Ensure all tests pass via `/verify_full_stack`.
- Update `CHANGELOG.md` with relevant phase information.
- Provide SHA-256 manifests for critical intellectual property updates.

*Stay sovereign.*
