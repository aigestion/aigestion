# Security Policy

## Supported Versions

We currently support the following versions of AIGestion with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of AIGestion seriously. If you believe you have found a security vulnerability, please do NOT open a public issue.

Instead, please report it via one of the following methods:

1.  **GitHub Private Vulnerability Reporting**: Use the "Report a vulnerability" button in the "Security" tab.
2.  **Email**: Send an email to [alejandro@example.com] (Placeholder) with the subject "SECURITY VULNERABILITY: [Brief description]".

Please include as much information as possible to help us reproduce and fix the issue. We aim to respond to all reports within 48 hours.
