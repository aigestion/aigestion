# Soporte para Usuarios Finales NEXUS V1 Agent

## ¿Qué es NEXUS V1 Agent?
Un asistente inteligente que responde preguntas, consulta el clima, y ayuda en tareas del proyecto NEXUS V1.

## ¿Cómo interactuar?
- Abre la consola y ejecuta:
  ```pwsh
  python agente_base.py
  ```
- Escribe tu pregunta y presiona Enter.
- Recibirás una respuesta directa y relevante.
- Puedes calificar la respuesta (1-10 o ok/nok) y sugerir mejoras.

## Funcionalidades principales
- Consulta de clima (real si está configurado)
- Respuestas en español, inglés o francés
- Sugerencias proactivas y explicaciones
- Aprendizaje automático de tus correcciones

## ¿Qué hacer si…?
- **No entiendes la respuesta**: Pide más detalles o aclara tu pregunta.
- **Quieres cambiar el idioma**: Solicítalo o pide ayuda en la consola.
- **Notas errores o incoherencias**: Usa el feedback para mejorar el agente.
- **No tienes permisos**: Contacta al administrador para cambiar tu rol.
