# Onboarding Guide

Welcome to NEXUS V1! This guide will help you get started as a new contributor or team member.

## 1. Prerequisites

- Node.js 18+, pnpm, Docker (for backend/k8s), Git

## 2. Setup

- Clone the repo and install dependencies:
  ```sh
  git clone https://github.com/noepab/NEXUS V1.git
  cd NEXUS V1
  pnpm install
  ```
- See `docs/` for architecture and feature docs

## 3. Development Workflow

- Use feature branches for changes
- Run tests and lint before PRs
- Use `pnpm dev` to start all apps/services

## 4. Community & Support

- Read the [Code of Conduct](./CODE_OF_CONDUCT.md)
- Join our chat/forum (link in README)
- For help, open an issue or ask in the community
