## ¿Cómo contribuir?

1. Crea un branch desde `main`.
2. Sigue las guías de PR y issues.
3. Asegúrate de pasar lint, test y coverage.
4. Solicita revisión a @noepab.

## Código de conducta

- Respeto y colaboración.
- Documenta tus cambios.
- Usa los templates de PR/issue.
