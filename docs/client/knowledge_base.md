### How do I access the Admin Dashboard?
The Admin Dashboard is accessible via `https://aigestion.net/admin` or locally at `http://localhost:3000/admin` (if authorized).

### What is 'God Mode'?
God Mode is our highest level of administrative access, allowing for deep system audit, real-time remediation, and full infrastructure control.

### How are payments handled?
We use Stripe for automated subscription billing and user provisioning. Checkout occurs via our secure `aigestion.net/billing` portal.
