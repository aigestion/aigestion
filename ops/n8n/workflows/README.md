# n8n Workflows

This directory contains 40+ automated workflows for **n8n**, covering Webhooks, Business Logic, AI/ML Pipelines, and Ops/Maintenance.

## How to Import
1. Open your n8n dashboard.
2. Create a new Workflow.
3. Click the **top-right menu** (three dots) -> **Import from File**.
4. Select one of the `.json` files in this directory.
5. Save and Activate.

## Configuration
Most workflows use **Environment Variables** (e.g., `{{$env["SLACK_WEBHOOK_URL"]}}`) or placeholders.
Ensure these variables are set in your n8n environment variables or `.env` file before activating.
