# Database Maintenance Workflow

This workflow manages the backup and restoration of data persistence layers.

## workflows

### Backup
1. **Redis Backup**
   ```bash
   docker exec -t redis-container redis-cli save
   docker cp redis-container:/data/dump.rdb ./backups/redis/dump_$(date +%Y%m%d).rdb
   ```
2. **Postgres/SQL Backup** (if applicable)
   ```bash
   docker exec -t postgres-container pg_dumpall -c -U postgres > ./backups/postgres/dump_$(date +%Y%m%d).sql
   ```

### Restore
1. **Redis Restore**
   - Stop Redis container.
   - Replace `/data/dump.rdb` with backup file.
   - Start Redis container.

2. **Verified Restore Check**
   - Run a query to verify data integrity after restore.
