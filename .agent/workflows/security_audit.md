# Security Audit Workflow

This workflow identifies vulnerabilities in dependencies and code.

## Steps

### 1. Dependency Audit

// turbo

1. Audit Node.js dependencies (Root).
   ```bash
   pnpm audit --audit-level moderate
   ```
   // turbo
2. Audit Backend dependencies.
   ```bash
   cd backend
   pnpm audit --audit-level moderate
   ```
   // turbo
3. Audit Frontend dependencies.
   ```bash
   cd frontend
   pnpm audit --audit-level moderate
   ```
   // turbo
4. Audit Python dependencies (ML Service).
   ```bash
   cd ml-service
   pip install safety
   safety check -r requirements.txt
   ```

### 2. Static Analysis


### 3. Reporting

6. Generate a summary report in `logs/security_audit_[DATE].log`.
